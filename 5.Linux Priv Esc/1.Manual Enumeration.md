## Active Network Connections and Listening Ports

```bash
ss -anp

# See something interesting? Example:
tcp                    LISTEN                  0                       511                                                                       127.0.0.1:8000                                                 0.0.0.0:*

# See what it is:
curl http://127.0.0.1:8000

# Website? Check common file locations
ls /var/www/html
ls /srv/http
ls /opt

```
