## Automating Recon

### Summary
Automating recon uses tooling to consistently collect common recon data (headers, SSL, WHOIS, DNS, subdomains, crawling, directories, Wayback, ports) faster and at scale.

------------------------------------------------------------

## EXERCISE: FinalRecon Setup

### Task 1: Install FinalRecon

1. Run: `git clone https://github.com/thewhiteh4t/FinalRecon.git`
   - Target-agnostic: `git clone <FINALRECON_GIT_URL>`
   - Example result:
     - `Cloning into 'FinalRecon'...`

2. Run: `cd FinalRecon`
   - Target-agnostic: `cd FinalRecon`

3. Run: `pip3 install -r requirements.txt`
   - Target-agnostic: `pip3 install -r requirements.txt`

4. Run: `chmod +x ./finalrecon.py`
   - Target-agnostic: `chmod +x ./finalrecon.py`

5. Run: `./finalrecon.py --help`

### Key Takeaway
- FinalRecon bundles multiple recon steps into one tool and can be run modularly (`--headers`, `--whois`, etc.) or as a full sweep (`--full`).
