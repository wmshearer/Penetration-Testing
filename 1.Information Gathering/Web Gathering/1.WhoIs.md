## WHOIS

### Summary
WHOIS is a query protocol used to retrieve registration and ownership details for domains, IP ranges, and autonomous systems, commonly used during reconnaissance to assess legitimacy, infrastructure, and potential risk indicators.

------------------------------------------------------------

## WHOIS Enumeration

### Task 1: Install WHOIS (if not installed)

1. Run: 
   ```
   sudo apt update
   ```
   
2. Run: 
   ``` 
   sudo apt install whois -y
   ```
   - Example result: `Setting up whois (5.5.17) ...`

------------------------------------------------------------

### Perform a WHOIS lookup on a domain

1. Run: `whois inlanefreight.com`
   - Target-agnostic: `whois <DOMAIN>`
   - Example result:
     - `Domain Name: inlanefreight.com`
     - `Registrar: Amazon Registrar, Inc.`
     - `Creation Date: 2019-08-05`
     - `Updated Date: 2023-07-03`

------------------------------------------------------------

### Key Data Extracted from WHOIS
- Domain registration and expiry dates
- Registrar and abuse contact information
- Registrant organization and contacts (if not privacy-protected)
- Name servers and DNS ownership
- Domain status flags (transfer/update protections)

------------------------------------------------------------
