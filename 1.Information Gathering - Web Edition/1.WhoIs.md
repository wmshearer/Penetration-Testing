## WHOIS

### Summary
WHOIS is a query protocol used to retrieve registration and ownership details for domains, IP ranges, and autonomous systems, commonly used during reconnaissance to assess legitimacy, infrastructure, and potential risk indicators.

------------------------------------------------------------

## EXERCISE: WHOIS Enumeration

### Task 1: Install WHOIS (if not installed)

1. Run: `sudo apt update`
   - Target-agnostic: `sudo apt update`
   - Example result: `Reading package lists... Done`

2. Run: `sudo apt install whois -y`
   - Target-agnostic: `sudo apt install whois -y`
   - Example result: `Setting up whois (5.5.17) ...`

------------------------------------------------------------

### Task 2: Perform a WHOIS lookup on a domain

1. Run: `whois inlanefreight.com`
   - Target-agnostic: `whois <DOMAIN>`
   - Example result:
     - `Domain Name: inlanefreight.com`
     - `Registrar: Amazon Registrar, Inc.`
     - `Creation Date: 2019-08-05`
     - `Updated Date: 2023-07-03`

------------------------------------------------------------

### Task 3: Perform WHOIS lookup on a well-known domain

1. Run: `whois facebook.com`
   - Target-agnostic: `whois <DOMAIN>`
   - Example result:
     - `Registrar: RegistrarSafe, LLC`
     - `Creation Date: 1997-03-29`
     - `Registry Expiry Date: 2033-03-30`
     - `Registrant Organization: Meta Platforms, Inc.`
     - `Name Server: A.NS.FACEBOOK.COM`

------------------------------------------------------------

### Key Data Extracted from WHOIS
- Domain registration and expiry dates
- Registrar and abuse contact information
- Registrant organization and contacts (if not privacy-protected)
- Name servers and DNS ownership
- Domain status flags (transfer/update protections)

------------------------------------------------------------

### Key Takeaway
- Recently registered domains + privacy protection + suspicious name servers often indicate phishing or malicious infrastructure.
- WHOIS is most effective when combined with DNS, IP, and hosting enumeration techniques.
